# Product Monitoring System

Automated product monitoring and tracking system for marketplace and e-commerce platforms.

The system monitors listings, tracks updates, collects data, and helps automate repetitive monitoring workflows.

## Features

- Automated product monitoring
- Listing tracking
- Data collection and processing
- Browser automation
- Monitoring workflows
- Flexible filtering

## Tech Stack

- Python
- Playwright
- Telebot

## Installation

```bash
git clone https://github.com/vscerbaks/product-monitoring-system.git
cd product-monitoring-system

pip install -r requirements.txt
```

## Usage
```python main.py```
